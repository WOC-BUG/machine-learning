# 3.3对数几率回归（逻辑回归）

## 问题描述
回归问题变为分类问题，x的取值只有几种，y=0,1,2这种
考虑广义线性模型，只需要用一个单调可微的关系函数将分类任务的标记y做一个转换就可以了

## 二分类任务
y={0,1}

### 关系函数

**单位阶跃函数(unit-step function)：**

$$ y=\left\{
\begin{aligned}
\text{0, x<0} \\
\text{0.5,x=0} \\
\text{1,x>0}
\end{aligned}
\right.
$$

**对数几率函数(logistic function Sigmoid)：**

替换为单调可微的函数
![avatar](\对数几率函数图.png)

**函数式：**
$y=\frac{1}{1+e^{-z}}=\frac{1}{1+e^{-(w^Tx+b)}}$
$ln\frac{y}{1-y}=w^Tx+b$
其中$\frac{y}{1-y}$是几率(odds)：x取1的相对可能性，对数几率(log odds)为$ln\frac{y}{1-y}$

**本质：**
用线性回归模型的结果取逼近真实标记的对数几率

**优点：**
1. 直接对分类可能性进行建模
2. 无需事先假设数据分布，避免了假设分布不准确带来的问题
3. 不只是预测出类别，而实得到近似概率推测，对许多需要利用概率辅助决策的任务很有帮助
4. 任意阶可导的凸函数，有很好的数学性质
5. 现有的许多数值优化算法都可直接用于求取最优解
